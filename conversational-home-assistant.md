# Home Assistant Configuration

Configure the complete voice assistant pipeline in Home Assistant.

## Prerequisites

- Lemonade Server running with all models configured
- Home Assistant 2024.12+ installed
- HACS plugin installed

See [LLM Configuration](/llm-configuration) and [Embedding Model](/embedding-model) for model setup.

## Pipeline Components

1. **STT Endpoint** - Speech-to-text via Whisper
2. **LLM Endpoint** - Language understanding via Qwen/gpt-oss
3. **TTS Endpoint** - Text-to-speech via Kokoro
4. **Embedding Endpoint** - Semantic search via nomic
