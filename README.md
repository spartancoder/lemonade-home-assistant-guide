# Lemonade Home Assistant Guide

Complete guide to setting up a local voice assistant with Lemonade Server.

## Large Files

Configuration files and scripts are distributed via GitHub Releases to avoid the Git LFS limitation with GitHub Pages.

**Status:** Release v1.0.0 is active and ready for file uploads in Phase 3.

**Download Pattern:**

```text
https://github.com/spartancoder/lemonade-home-assistant-guide/releases/download/v1.0.0/{filename}
```

See: [Releases](https://github.com/spartancoder/lemonade-home-assistant-guide/releases)
