# Conversational Setup

Configure the language model and embedding model for intelligent conversation.

## Overview

This section covers:

- LLM Configuration (GPU and NPU options)
- Embedding Model setup
- Home Assistant integration

## Components

- **LLM Configuration** - Set up Qwen3-4B or gpt-oss-20b for language understanding
- **Embedding Model** - Configure nomic-embed-text for semantic search
- **Home Assistant** - Connect everything to your voice assistant pipeline
