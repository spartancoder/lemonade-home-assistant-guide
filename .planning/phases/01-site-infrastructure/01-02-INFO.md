## Large File Distribution

**Decision:** Using GitHub Releases instead of Git LFS

**Rationale:**
- Git LFS cannot be used with GitHub Pages (official GitHub limitation)
- GitHub Releases provides free file hosting with versioning
- Direct download links can be embedded in documentation
- Integrated with GitHub repository (no external service needed)

**Usage:**
- Configuration files and scripts will be attached to GitHub releases
- Documentation will link to release assets using stable URLs
- Each tutorial section will reference specific release versions for file downloads

**Release Pattern:**
- v1.0.0: Initial configuration files and scripts (created in Phase 1)
- v1.1.0, v1.2.0, etc.: Future updates as needed

**Download URL Pattern:**
```
https://github.com/spartancoder/lemonade-home-assistant-guide/releases/download/v1.0.0/filename.conf
```
