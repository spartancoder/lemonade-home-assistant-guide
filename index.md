# Lemonade Home Assistant Guide

Complete guide to setting up a local voice assistant with Lemonade Server and

## Overview

This tutorial will walk you through:

1. Installing Lemonade Server
2. Setting up AI models for speech-to-text, text-to-speech, and language understanding
3. Integrating with Home Assistant
4. Configuring the complete voice assistant pipeline

## Prerequisites

1.  A computer capable of running lemonade-server
    - AI models are memory hogs, the more memory the better.
    - Supported Operating Systems
        - Linux - at least 32GB of memory
        - Windows - at least 48GB of memory
2.  Voice Hardware
    - This is what we will use to replace Alexa, Google Home, and Siri.
    - These come in many different sizes and price points.
    - If you do not have this currently, you can use "Assist for Android/Apple" as to test the voice pipeline in this tutorial.
